# Quickstart
```bash
nvm install 10.15.3;
nvm use 10.15.3;
npm rebuild node-sass;
cd ~/sunpower/energylink-connect2/energylink-connect2-app;
npm i;
npm run build;
cd ~/sunpower/energylink-connect2/
npm i;
phongap init ios;
phonegap init android;
phonegap prepare ios;
phonegap prepare android;
```

## Run in iOS simulator
```bash
cd ~/sunpower/energylink-connect2/;
cordova emulate ios;
```
